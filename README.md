نبذة عن المشروع
هذا المشروع عبارة عن تحليل بيانات متكامل لشركة اتصالات، يهدف إلى فهم أسباب ترك العملاء للخدمة (Customer Churn) وتأثير ذلك على الإيرادات. تم بناء داشبورد تفاعلي احترافي يوضح أهم المؤشرات واتجاهات البيانات بشكل مرئي وسهل الفهم.
🎯 أهداف المشروع
- تحديد نسبة العملاء الذين تركوا الخدمة وأسباب ذلك
- تحليل الإيرادات وعلاقتها بسلوك العملاء
- فهم الفئات الأكثر عرضة للـ Churn
- تقديم رؤى قابلة للتطبيق لتحسين الاحتفاظ بالعملاء
## 🛠️ الأدوات والتقنيات المستخدمة

| الأداة | الاستخدام |
| **Python** | تنظيف البيانات، التحليل الإحصائي، الـ Visualizations |
| **Pandas** | معالجة وتحليل البيانات |
| **Matplotlib / Seaborn** | رسم الجراف والمخططات التحليلية |
| **SQL** | استخراج البيانات وتصفيتها وتجميعها |
| **Power BI** | بناء الداشبورد التفاعلي الاحترافي |

- الفئة أقل من 20 سنة الأقل في معدل الـ Churn
### 2. تحليل طرق الدفع
